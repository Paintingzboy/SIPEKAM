# SIPEKAM
Developed an IoT-based smart agricultural solution designed to optimize the post-harvest rice drying process for local farmers. The system integrates renewable energy with real-time monitoring to ensure high-quality rice production

![Image Alt](https://github.com/Paintingzboy/SIPEKAM/blob/97ae19c8bd66c96b4f403b4d10dd10daf8541248/SIPEKAM.png)

Key Features & Technical Implementation:
- IoT Integration: Utilized ESP32 to monitor real-time temperature and humidity via DHT22 sensors, accessible through a custom-built web dashboard.
- Renewable Energy: Powered by a Solar Power System (Solar Panels, SCC, and Lead-Acid Battery) with an LM2596 Buck Converter for stable voltage regulation.
- Automated Control: Implemented a dual-channel relay system to control heating lamps (parallel circuit) and exhaust fans for optimal airflow and moisture removal.
- User Interface: Designed an interactive web-based dashboard and on-device OLED display for local and remote monitoring.
